# DataStructures

A collection of data structure implementations and algorithms in C#.

## Description

This project contains solutions to common data structure and algorithm problems, implemented in C#. It includes the main library project and a corresponding test suite.

## Project Structure

- `DataStructures/`: Main project containing data structure implementations
  - `GroupAnagrams.cs`: Solution for grouping anagrams
  - `ThreeSum.cs`: Solution for the three sum problem
- `DataStructuresTest/`: Unit tests for the data structures

## Prerequisites

- .NET 10.0 SDK or later

## Building the Project

To build the solution:

```bash
dotnet build DataStructures.sln
```

## Running Tests

To run the unit tests:

```bash
dotnet test DataStructures.sln
```

## Usage

Add a reference to the `DataStructures` project in your application and use the provided classes and methods.
